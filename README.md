# treeapp
Faültető App egy böngészőben futó közösségi játék, ahol coinokért fákat ültethetsz, élő ranglistán versenyezhetsz, és valós időben figyelheted a közösség aktivitását. Animációk, mobilbarát dizájn és élő feed teszik izgalmassá. A cél: játékosan építeni egy zöldebb jövőt. Készítette: Székely György.
